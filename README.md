# Sample Website

This is a sample website made as part of *Learn Enough Git to Be Dangerous*, possibly the greatest beginner Git tutorial ever written.  
You should totally [check it out](https://www.learnenough.com/git-tutorial),  
and be sure to join the email list and follow [@learnenough](https://twitter.com/learnenough) on Twitter.

After finishing *Learn Enough Git to Be Dangerous*, you'll know enough Git to be **dangerous**.  
This means you'll be able to track changes in your projects, back up data, share your work with others,  
and collaborate with programmers and other users of Git.
